Дополнительное задание:

Создан файл для cloud-init для автоматической установки пакетов и деплоя приложения после создания вм.

Команда для создания вм  YC CLI с последующим запуском скрипта startup:

yc compute instance create \
  --name reddit-app \
  --hostname reddit-app \
  --memory=4 \
  --create-boot-disk image-folder-id=standard-images,image-family=ubuntu-1604-lts,size=10GB \
  --network-interface subnet-name=default-ru-central1-b,nat-ip-version=ipv4 \
  --metadata serial-port-enable=1 \
  --metadata-from-file user-data=./startup
